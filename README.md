# Financial, Operational, and Market Performance of Romanian Retail Companies (2015‚Äì2024)

This project analyzes the **financial, operational, and market performance** of the main retail companies in Romania between **2015‚Äì2024**, using **Power BI dashboards** built on official and processed data.  

The analysis integrates **financial indicators**, **operational data**, and **macroeconomic context**, offering both an overview and advanced insights into the retail sector.

## Key Features
- **Financial indicators**: turnover, net profit, net margin, ROE, debt ratio.  
- **Operational indicators**: number of stores, number of employees.  
- **Macroeconomic context**: GDP, inflation, unemployment.  
- **Interactive dashboard**: filters by company and year, advanced visualizations (scatter plots, box & whisker, double line charts).

## Main Insights
- **Market Leaders**: Lidl and Kaufland dominate in turnover, while Dedeman achieves the highest profitability (net margin & ROE).  
- **Expansion Trends**: steady growth in turnover for top 5 companies, with different strategies across players.  
- **Operational Efficiency**: Dedeman shows high turnover per store, while Profi faces efficiency challenges.  
- **Macroeconomic Links**: retail growth strongly correlates with GDP and consumer purchasing power.

## How to Use
1. Clone/download the repository.  
2. Open the `dashboard.pbix` file in **Power BI Desktop**.  
3. Use the filters (Company, Year) to explore the data interactively.
